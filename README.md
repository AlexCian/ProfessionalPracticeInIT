# Professional Practice In IT (Fletnix)

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.1.5.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Navigation

The header up top has all of the desired links for you to go to on the website in button forms.

## Main Project

The project runs off the server and is fully loaded on startup (no loading time between pages).

This project is an Angular project that is a movie and TV program watchlist. It is a full website with a home page, featured programs, items to purchase such as boxsets, a fully functioning cart page and a support page to view the FAQ and send off forms for technical assistance. It is mainly for broadcasting and previews of popular TV shows (an informative website) with some functionality.
